2CPay is a 2Checkout plugin developed for Woocommerce

== Description ==
2CPay is a wordpress plugin which provides payments via 2Checkout for Woocommerce stores. This plugin provides payments via hosted page.

== Installation ==
1. Upload the plugin files to the '/wp-content/plugins/2cpay' directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Click on Settings on Plugins screen OR go to Woocommerce->Settings->Checkout->2CPay.
4. Now put your 2Checkout Store Id(i.e. Account No.) and Secret Word in the form and enable the payment gateway.
5. Login to your 2Checkout account and change the settings Account->Site Management->Checkout Options->Direct Return to 'Given links back to my website'

== Frequently Asked Questions ==
